Number of iterations between measurements.

◆ measurement_time_interval

unsigned long long measurement_device_eml::measurement_time_interval

Estimated amount of time in between measurements.

Used to adjust the library to the actual physical device being used to measure energy. Defaults to 150ms, but have to be adjusted to the physical device in order to improve the dynamic load balancing procedures.
