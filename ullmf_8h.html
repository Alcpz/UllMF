Configures a ullmf_calibration_t, the data structure that controls workload distribution, workload minimum size (blocksize) and workload distribution strategy.

It also defines the root process and the MPI Communicator.

Parameters
[in]calibullmf_calibration_t control data structure to initialize.
[in]countsAmount of work assigned to each process.
[in]displsDisplacement vector with the location of the counts assigned to each process.
[in]blocksizeMinimum step to move counts between process (amount of counts per unit of work).
[in]strategyChosen strategy to perform the dynamic load balancing.
[in]rootProcess assigned to manage the load balancing logic.
[in]commMPI Communicator to be used inside the dynamic load balancing procedures.
Return values
ULLMF_SUCCESSif the setup was performed successfully.

enum ullmf_error ullmf_mpi_stop ( ullmf_calibration_t *  calib,
int *  counts,
int *  displs 
)

Stops the calibration phase.

Uses the measurements gathered since the last ullmf_mpi_start call to perform the dynamic load balancing strategy defined in the setup phase. Determines the new workload distribution, which gets returned inside counts and displs.

Parameters
[in]calibullmf_calibration_t control data structure.
[out]countsAmounts of data assigned to each process.
[out]displsDisplacements Vector from the data base address for each process.
Return values
ULLMF_SUCCESSif the calibration process was performed successfully.
